Port to Windows Store of [Simon Tatham's Portable Puzzle Collection](http://www.chiark.greenend.org.uk/~sgtatham/puzzles/).

Version 1.1 features 9 puzzles : 
* Pattern
* Map
* Lightup
* Net
* Bridges
* Same Game
* Signpost
* Slant
* Untangle

Note: Source have been transformed into plain C#.
